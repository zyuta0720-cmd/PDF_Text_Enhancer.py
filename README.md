# PDF_Text_Enhancer.py

使い方
- python PDF_Text_Enhancer.py を実行します。
- 「PDFを開く」ボタンから対象のファイルを選択します。
- 「強調度」スライダーを動かして、文字が一番綺麗に見えるポイントを探します。
- お好みの加工モードを選択します。
- 「保存」ボタンを押すと、全ページに処理を適用したPDFが出力されます。
